Simulação de Infraestrutura de Rede para Universidade

Este projeto envolveu a criação de uma simulação de uma infraestrutura de rede completa para uma universidade, utilizando o Cisco Packet Tracer. A rede foi projetada para suportar conectividade eficiente e segura entre diferentes departamentos e áreas da instituição.

Principais componentes incluem:

Topologia de Rede:Configuração de uma rede hierárquica com switches, roteadores e dispositivos finais.
Endereçamento IP: Implementação de sub-redes para gerenciamento de tráfego e segurança.
Segurança:Uso de VLANs e ACLs para proteger a rede e segmentar o tráfego.
Serviços de Rede: Configuração de DHCP, DNS, e servidores web e de e-mail.
Gerenciamento de Tráfego: Implementação de roteamento dinâmico e QoS para priorizar tráfego crítico.
